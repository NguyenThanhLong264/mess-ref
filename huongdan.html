Hướng dẫn tạo link chat Messenger có kèm ref

🎯 Mục tiêu

Tạo link chuyển sang cửa sổ trò chuyện Messenger và gửi kèm tham số ref là URL hiện tại để theo dõi nguồn gốc người dùng.

⚙️ Cách làm

Chúng ta sẽ dùng trực tiếp link Messenger theo định dạng:

https://m.me/username?ref=tham-so

📌 Các bước thực hiện

🔹 Bước 1: Tạo Fanpage và lấy link m.me

Vào phần giới thiệu fanpage để lấy username. Link dạng:

https://m.me/yourpage

🔹 Bước 2: Tạo đoạn HTML để chèn vào website

Dùng đoạn mã sau:

<a href="https://m.me/yourpage?ref=" + encodeURIComponent(window.location.href) target="_blank">
  Nhắn tin qua Messenger
</a>

Hoặc dùng thẻ `script` để tạo tự động:

<script>
  const currentUrl = encodeURIComponent(window.location.href);
  const chatLink = "https://m.me/yourpage?ref=" + currentUrl;
  document.write('<a href="' + chatLink + '" target="_blank">Chat qua Messenger</a>');
</script>

🔹 Bước 3: Gắn tham số theo dõi (ref)

Phần `ref` sẽ nhận URL hiện tại để bạn biết người dùng đến từ trang nào.

Ví dụ nếu người dùng đang ở:

https://example.com/blog/123

Thì link chat tạo ra sẽ là:

https://m.me/yourpage?ref=https%3A%2F%2Fexample.com%2Fblog%2F123 Dùng thử link mẫu
